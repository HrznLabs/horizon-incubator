## 2024-05-22 - Documentation as UX
**Learning:** In non-code repos, the "interface" is the documentation structure. Adding TOCs and proper semantics directly improves the user journey.
**Action:** Always check for long markdown files without TOCs as a quick win.

## 2025-12-05 - Semantic Markdown for ASCII Art
**Learning:** Generic code blocks (```) for ASCII art can cause distracting syntax highlighting artifacts. Using explicit `text` tags improves visual stability and readability.
**Action:** Always tag ASCII diagrams with `text` or `plaintext`.
